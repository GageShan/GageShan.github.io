一只小蜜蜂... HDU - 2044 (简单dp)

题目

有一只经过训练的蜜蜂只能爬向右侧相邻的蜂房，不能反向爬行。请编程计算蜜蜂从蜂房a爬到蜂房b的可能路线数。
其中，蜂房的结构如下所示。


Input
输入数据的第一行是一个整数N,表示测试实例的个数，然后是N 行数据，每行包含两个整数a和b(0<a<b<50)。
Output
对于每个测试实例，请输出蜜蜂从蜂房a爬到蜂房b的可能路线数，每个实例的输出占一行。

Sample Output

思路

由蜂房的结构可以看到，蜂房1可以直接到达蜂房3，蜂房2也可以直接达到蜂房3，那么到达蜂房3的路线为蜂房1的路线加蜂房2的路线，递推公式

#include
using namespace std;
int main(){
int n;
long long int dp[550];
dp[1]=1;
dp[2]=1;
cin>>n;
while(n–){
int a,b;
cin>>a>>b;
dp[a]=1;
dp[a+1]=1;
for(int i=a+2;i<=b;i++)
dp[i]=dp[i-1]+dp[i-2];
cout<<dp[b]<<endl;
}
return 0;
}
`
