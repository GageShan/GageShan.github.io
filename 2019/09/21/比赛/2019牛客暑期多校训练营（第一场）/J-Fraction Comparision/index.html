2019牛客暑期多校一 J-Fraction Comparision（取模技巧）

题目

Bobo has two fractions ${x} \over {a}$ and ${y} \over {b}$. He wants to compare them. Find the result.
输入描述:
The input consists of several test cases and is terminated by end-of-file.

Each test case contains four integers x, a, y, b.

  • 0≤x,y≤1018
  • 1≤a,b≤109
  • There are at most 105105 test cases.

输出描述:
For each test case, print = if ${x} \over {a}$=${y} \over {b}$. Print < if x/a<y/b. Print > otherwise.

输出

思路

${x} \over {a}$ 和 ${y} \over {b}$比较大小，可以转化为x*b和y*a比较大小。
而数据是很大的，直接乘肯定会爆精度，所以可以将x*b转化为n\*10<sup>9</sup>+c这样一种形式。将两者都转化为这样的，就可以通过比较n和c的值来比较两者大小。不过要注意的是，n1大于n2时不能说明x*b就一定大于y*a，因为当n1=2，n2=1，而c1=3*109，c2=109时，实际上前者小于后者，所以就要把c1和c2的值除以109放入n1和n2中，再比较。
